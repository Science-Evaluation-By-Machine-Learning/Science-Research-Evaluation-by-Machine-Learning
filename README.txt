项目愿景：

1.现在的中国科研评价体系有很大的问题，唯论文的评价体系，造就了很多专门为了发论文的灌水者，这部分人造成了劣币驱逐良币，破坏整个学术氛围，生产出来一堆无用的水论文，所以必须通过一种工具来约束人性，重新制定评价体系，更客观，更真实。

2.人的寿命是有限的，最近两百年知识大爆炸，需要记忆学习的知识数据越来越多。难免造成人脑容量不够用，很怀疑再过200年，人类还适不适合做研究工作，而且即使即使某一个研究工作者能力很强，比如爱因斯坦，可是他也最多活到80岁就去世了，他的大脑不能继续为人类服务，且他真正有效工作的时间就是中青年时期，因此为了将来知识大爆炸，有必要发明一种AI科学家系统，来辅助甚至未来取代人类来做科研工作

3.希望像淘宝每年的双十一成交额显示屏中展示的一样，当某个学科或研究方向出现更新时，就会在屏幕上亮起一个点，或一条连接线，整个世界的知识图谱又更新了，以后大家就是辛勤的挖掘知识宝库的挖矿工，负责点亮知识图谱业务，以后的科研经费可以通过这套系统来分配，甚至可以在这套系统上雇佣非人类人工智能科学家（电脑系统），来做知识矿工，底层的研究者可以做实验的实际操作者，高层的研究者可以做更尖端，更困难的研究。
